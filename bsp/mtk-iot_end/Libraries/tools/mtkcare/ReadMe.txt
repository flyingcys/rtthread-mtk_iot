How to Build MtkCare Demo APK?
1. Select and open Code/MtkCare project in Android Studio welcome GUI.
2. Select "OK" for Gradle Sync Dialog: "Would you like the project to use hte Gradle wrapper".
3. Set careDemo module as current app module;
   Set Build Variants of careDemo module as release;
4. Keep network availabe.
   If you are in Chinese Mainland, your network must could over the wall. (can access "facebook" web)
5. Build APK, then you will get the APK in "Code\MtkCare\careDemo\build\outputs\apk" folder.

For Andrid Studio problem, you could refer to web (https://developer.android.com/studio/intro/index.html).
